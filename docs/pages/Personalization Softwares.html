Stealth Assistant

Stealth Assistant is a virtual assistant for Microsoft Windows 10. It works as a good alternative for Cortana.

It can do various tasks like:

  • Opening files, apps or a web result.
  • Search anything locally or on the internet.
  • Get you weather data.
  • Perform general tasks and operations.
  • And many more...

Publisher: StealtherThreat

Compatibility: Windows 10

Minimum System Requirement:

  • Processor - Quad Core processor
  • Ram - 3GB

    • Note - Although it can run on any dual core CPU with 2GB of Ram, but audio may not work properly on doing so.

Recommended System Requirement:

  • Ram - 4GB
  • Storage - Any SSD
  • A high speed internet Connection
